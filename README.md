# БВ Помощник

### RAG-ассистент по конкурсу научно-технологических проектов школьников "Большие вызовы" в Новосибирской области

Проект представляет собой MVP справочного ассистента, который отвечает на вопросы по базе знаний и показывает источники ответа.

Демо доступно по ссылке: https://big-challenges-rag-assistant-54.streamlit.app/

### О проекте
Ассистент помогает разобраться в базовых вопросах о конкурсе: кто может участвовать, какие есть форматы конкурса, чем основной конкурс отличается от "ПРО Больших вызовов", что нужно указать в заявке, какие есть направлени и где лучше уточнять актуальную инофрмацию.

Проект не является официальным источником информации и создан для снижения нагрузки на координаторов конкурса.

### Как работает ассистент
Пользователь задает вопрос, система ищет релевантные фрагменты в базе знаний, передает найденный контент в LLM и формирует ответ на основе этих источников.
После ответа ассистент показывает, какие разделы базы знаний были использованы.

В проекте реализован гибридный формат: поиск производится как по эмбеддингам, так и по ключевым словам.

### Технологии
Python, Streamlit, Sentence Transformers, scikit-learn, Hugging Face Inference API, Markdown, Streamlit Community Cloud.

В проекте была собрана база знаний по конкурсу, реализован RAG-пайплайн, добавлен поиск по смысловым разделам, настроена генерация ответа через Hugging Face API, сделан Streamlit-интерфейс и выполнено ручное тестирование типовых и рискованных вопросов.


### Статус проекта
Проект реализован как MVP и опубликован в виде Streamlit-приложения. Происходит обсуждение о доработке и внедрении ассистента на сайт "Больших вызовов" в Новосибирской области.
